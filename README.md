# Student Management System (Java)

A console-based application to manage student records using Java and MySQL.

##  Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student

##  Tech Stack

* Java
* JDBC
* MySQL

##  Concepts Used

* Object-Oriented Programming (OOP)
* DAO Design Pattern
* Database Connectivity

##  Setup Instructions

1. Install MySQL

2. Create database:
   CREATE DATABASE student_db;

3. Create table:
   CREATE TABLE students (
   id INT PRIMARY KEY,
   name VARCHAR(50),
   department VARCHAR(50),
   email VARCHAR(50),
   phone VARCHAR(15)
   );

4. Update DB credentials in DBConnection.java

5. Run Main.java
